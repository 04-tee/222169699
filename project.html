Smart Lecturer Scheduling, Availability & Navigation System

Faculty of Natural Sciences, Mathematical Sciences and Computing Department

Walter Sisulu University

Project Overview

The Smart Lecturer Scheduling, Availability and Navigation System is an innovative mobile application designed to streamline the appointment booking process between students and faculty members at Walter Sisulu University. This system eliminates the challenges of manual scheduling and provides a seamless, user-friendly experience for both students and lecturers.

Problem Statement

The traditional process of scheduling appointments with lecturers can be time-consuming and frustrating for students. Common challenges include:

  • Difficulty in finding lecturers on campus
  • Lack of real-time availability information
  • Manual scheduling processes that lead to double-bookings
  • No centralized system for managing appointments
  • Time wasted searching for lecturer offices
  • Inefficient communication between students and faculty

Project Objectives

The main objectives of this project are:

  1. Develop an intuitive mobile application for appointment scheduling
  2. Implement real-time availability tracking for lecturers
  3. Provide indoor navigation to lecturer offices
  4. Enable students to view and book available time slots
  5. Allow users to manage and cancel appointments easily
  6. Reduce administrative burden on faculty members
  7. Improve communication between students and lecturers

Key Features

For Students:

  • Smart Scheduling: Browse available time slots and book appointments with ease
  • Real-Time Status: View lecturer availability (available, busy, in a meeting)
  • Indoor Navigation: Get directions to lecturer offices within the campus
  • Appointment Management: Cancel or reschedule appointments as needed
  • Notifications: Receive reminders about upcoming appointments

For Lecturers:

  • Availability Management: Set and update availability status in real-time
  • Appointment Overview: View all scheduled appointments in one place
  • Calendar Integration: Sync with existing calendar systems
  • Student Information: Access relevant student details for scheduled meetings
